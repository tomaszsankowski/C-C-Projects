# Distributed Processing

A collection of laboratory assignments focusing on distributed processing, operating systems concepts, and concurrent programming. The projects are implemented in C and Java.

They include:

- **Lab 1:** Weird priority queue (C)
- **Lab 2:** Static and dynamic libraries (C)
- **Lab 3:** Forks and semaphores (C)
- **Lab 4:** Windows processes and threads (C)
- **Lab 5:** Windows client-server (C)
- **Lab 6:** Multi-threading in Java (Java)
- **Lab 7:** Pipes on Linux (C)
- **Lab 7:** POSIX threads (C)
