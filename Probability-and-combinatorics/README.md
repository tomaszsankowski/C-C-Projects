# Probability and Combinatorics

A collection of laboratory assignments in the field of probability and combinatorics. The projects are implemented in C, C++, and Python.

They include:

- **Lab 1:** Permutations and variations (C)
- **Lab 2:** Random number generators (C++)
- **Lab 3:** Random variables with arbitrary distribution (Python)
- **Lab 4:** Multidimensional random variables (Python)
