# Virtual World Simulator

A C++ project implementing a virtual world simulation based on the object-oriented programming paradigm.

Various organisms coexist in this world, including:
* **Animals:** Wolf, Sheep, Fox, Turtle, Antelope, Human.
* **Plants:** Grass, Dandelion, Guarana, Deadly Nightshade, Sosnowsky's Hogweed.

Each organism has its own unique behaviors and rules of interaction with other entities on the board.
