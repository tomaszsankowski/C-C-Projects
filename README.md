# C/C++ Projects Collection

This repository contains a collection of smaller programming projects, implemented mainly in C and C++.

Detailed information about each project can be found in the dedicated `README.md` files inside their respective directories:

- **[CSS](./CSS/)** - CSS Parser with custom data structure implementations.
- **[GIPF](./GIPF/)** - Implementation related to the GIPF board game.
- **[Probability-and-combinatorics](./Probability-and-combinatorics/)** - Laboratory assignments on probability and combinatorics.
- **[Virtual-World-Simulator](./Virtual-World-Simulator/)** - Virtual world simulator (object-oriented programming).
- **[ShortestPathFinder](./ShortestPathFinder/)** - Shortest path finder (Dijkstra's algorithm, custom data structures).
- **[distributed-processing](./distributed-processing/)** - Laboratory assignments on distributed processing, operating systems, and concurrent programming.
